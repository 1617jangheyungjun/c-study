## WriteLine과 Write의 차이

* **WriteLine은(는) 그 한줄에 써주는 것 이다. 따라서 WriteLine을 여러번 사용하면 파이썬의 print처럼 줄바꿈이 생긴다.**

* **Write은(는) 그 한줄에 써주는게아닌 이 문장을 써라라는 것 이다. 따라서 Write를 여러번 사용하면 한줄에 여러번 써진다.**
